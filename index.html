Matching non-linear element heights

Easily match-up heights across a set of non-linear elements.
Allows for multiple sets, with each set having a different height defined by a JavaScript injected inline min-height.
For use where Flexbox is too cumbersome, or cannot be retro-fitted.

CodePen demo: Matching element heights

Super small vanilla script: Approx 417 bytes minified and gzipped (694 bytes uncompressed). No dependencies.

Example

Two different heights set, one for the "Blah" (data-matchHeights="B"), another for the "Yada" (data-matchHeights="Y"):

Yada, yada, yada. Yada, yada, yada. Yada, yada, yada. Yada, yada, yada.

How it works

Add data attribute to the elements you wish to equalise and give each the same value: data-matchHeights="Y".
Ensure the elements each have display: block;.
Then run the script.

All elements with the same data attribute value have the same min-height applied.
The min-height value is equal to the tallest elements height and is recalculated when the browser is resized.
